Welcome

Quarenne is MIDI visualisation software for macOS, currently in the alpha phase of development.

Updates

  • May 21, 2023: Version 0.1 released
    • MIDI input picker
    • MIDI chord displayed
    • 3D models change to a random colour on MIDI note on
    • 3D models can be rotated using WASD keys
    • 3D models can be imported in .obj format
